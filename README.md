# CS2 Voice Overlay (VoiceBridge)

Realtime voice translation overlay for Counter-Strike 2 (team voice / callouts).

## Requirements

- Windows
- Python 3.10+ recommended

## Setup

```bash
pip install -r requirements.txt
```

## Run

Double-click `start.bat` or:

```bash
python main.py
```

## Version

See `VERSION.txt` (currently 1.0.2).
